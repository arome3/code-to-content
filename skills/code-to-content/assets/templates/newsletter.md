## Editor's Checklist (delete before sending)

**Before Writing:**
- [ ] Content curated (5-10 items collected)
- [ ] Featured item selected
- [ ] Opening angle identified

**While Writing:**
- [ ] Subject line under 50 chars
- [ ] Subject line creates curiosity or promises value
- [ ] Intro feels personal, not broadcast-y
- [ ] Each link has unique commentary (not just summary)
- [ ] One clear CTA in the closing
- [ ] Tone is consistent throughout

**Before Sending:**
- [ ] Preview in email client (mobile + desktop)
- [ ] All links tested and working
- [ ] Spell check complete
- [ ] Personalization tags working ({{first_name}})
- [ ] Unsubscribe link works
- [ ] Sent test to self
- [ ] Scheduled for optimal time
